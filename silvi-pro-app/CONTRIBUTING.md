# Contributing to Silvi Haul Rate Calculator Pro

Thank you for your interest! This is a **commercial software product**. 

## For Developers

If you're working on this codebase:

1. **Never commit API keys** - Use `.env` file (already in `.gitignore`)
2. **Test before committing** - Run `npm start` to verify changes
3. **Follow naming conventions** - Use camelCase for variables, PascalCase for components
4. **Comment complex logic** - Especially AI integration and calculation formulas
5. **Update documentation** - If you add features, update README.md

## Branch Strategy

- `main` - Production-ready code
- `develop` - Development branch
- `feature/*` - New features
- `bugfix/*` - Bug fixes
- `release/*` - Release preparation

## Code Quality

- Run linting before commits
- Ensure all features work on Windows, Mac, and Linux
- Test with both free and paid API tiers
- Verify license system functionality
